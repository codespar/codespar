## Prerequisites

Before following any guide, make sure you have:

1. **CodeSpar running** — via [Docker](/docs/configuration/docker) or [Railway](/docs/configuration/railway)
2. **Environment variables set** — at minimum `ANTHROPIC_API_KEY` and `GITHUB_TOKEN` (see [Configuration](/docs/configuration))
3. **At least one channel enabled** — Slack, Discord, Telegram, or WhatsApp (see [Configuration](/docs/configuration))

## Recommended Order

If you are setting up CodeSpar for the first time, follow these guides in order:

1. [Link a Repository](/docs/guides/link-repository) — connect your project
2. [Multi-Channel Setup](/docs/guides/multi-channel) — add your team's channels
3. [Review Pull Requests](/docs/guides/review-prs) — automate code review
4. [Deploy Pipeline](/docs/guides/deploy-pipeline) — set up deployment orchestration
5. [Multi-Tenant / Organizations](/docs/guides/multi-tenant) — (if managing multiple projects or teams)
