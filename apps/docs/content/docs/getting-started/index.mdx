# Getting Started

This section walks you through setting up CodeSpar from scratch. Whether you want a quick local demo or a full production deployment, start here.

## Quickstart

**Time: ~5 minutes.** Clone the repo, set your API key, connect a Slack workspace, and send your first `@codespar` command. No Docker required.

[Go to Quickstart →](/docs/getting-started/quickstart)

## Installation

Full installation guide covering all deployment methods — npm from source, Docker Compose, Railway one-click deploy, and building from source with Turborepo.

[Go to Installation →](/docs/getting-started/installation)

## First Agent

Step-by-step tutorial to deploy your first Project Agent. You will create a Slack app, link a GitHub repository, execute a coding task, and review a pull request — all from Slack.

[Go to First Agent →](/docs/getting-started/first-agent)
